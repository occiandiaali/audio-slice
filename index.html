💡 How It Works

1️⃣
Upload one or more audio files. Each appears in the file list.
2️⃣
Click + to add a file as a clip in the timeline. Add as many as you like.
3️⃣
Click a clip row to select it, then drag on its waveform to mark a region.
✂
Cut removes the selection. Keep trims everything else away.
↩
Each clip has its own undo history (up to 30 steps). Ctrl+Z undoes on the selected clip.
↕
Use ↑ ↓ arrows in the clip label to reorder clips before joining.
⚡
Join & Download merges all clips top→bottom into one WAV file.
⬇
Clip downloads only the currently selected clip as WAV.
